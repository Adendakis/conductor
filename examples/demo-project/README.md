# Demo Project

Minimal conductor project scaffold — generated by `conductor new-project`.

Shows the basic structure: pipeline YAML, agent registration, prompt templates.

## Files

- `pipeline.yaml` — one phase, one step, one agent
- `agents/__init__.py` — registers agents with conductor
- `agents/example_agent.py` — starter ToolExecutor to customize
- `agents/example_cao_using_agent.py` — boilerplate for delegating to a CAO tmux session
- `prompts/example.md` — prompt template with variable placeholders

## Run

```bash
cd examples/demo-project
conductor init
conductor serve --port 8080
conductor watch
```
